# Tasks-on-SVG
Завдання для учнів Webстудії ПДМ
Завдання з теми SVG
«SVG-іконка “Мішень”»  
Навчитися:  
•	працювати з viewBox;  
•	використовувати координати SVG;  
•	будувати елемент із простих фігур (circle);  
•	розуміти центр SVG.  
________________________________________
## Умова завдання  
Створи SVG-іконку мішені, яка складається з:  
1.	Зовнішнього кола (контур)  
2.	Середнього кола (заливка)  
3.	Центральної точки  
________________________________________
## Вимоги  
1. SVG-контейнер  
•	viewBox="0 0 100 100"  
•	розмір на сторінці — 200 × 200 px  
2. Розміри  
•	Центр усіх кіл: (50, 50)  
•	Радіуси:  
o	велике коло — 45  
o	середнє — 25  
o	мале — 5  
3. Оформлення  
•	зовнішнє коло — без заливки, з обводкою  
•	середнє — з кольоровою заливкою  
•	центральне — чорне  
Використовуйте тег:  
<circle cx="" cy="" r="">
